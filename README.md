# Пересдача JavaScript

## Выбранный уровень

**Уровень 2 — средний:** мини-приложение «Поиск и отображение данных с API».

## Что реализовано

- Работа с localStorage 
- Автоматическая загрузка при открытии страницы и кнопка «Загрузить снова».
- Вывод заголовка и текста поста в виде **карточек**; элементы создаются в JavaScript.
- Поле **поиска по заголовку** (фильтрация массива по введённой строке).
- Индикатор **состояния загрузки** и сообщение при **ошибке** сети или ответа сервера.
- Кнопка **«Сбросить фильтр»** (очищает поиск и показывает все загруженные посты).
- Кнопка **«Очистить результаты»** (убирает посты с экрана и очищает сохранённый список).

## Файлы

- `index.html` — разметка страницы.
- `styles.css` — оформление.
- `app.js` — логика загрузки, фильтрации и отображения.
- `KONSPEKT.txt` — подробный разбор кода построчно (для защиты/объяснения преподавателю).

## Как открыть

Откройте `index.html` в браузере (двойной щелчок или через «Открыть с помощью»). Нужен доступ в интернет для запроса к API.
